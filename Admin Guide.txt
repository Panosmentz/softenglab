To connect use the following steps : 
1)Open a MySQL platform(Navicat) and XAMPP and start MySQL
2)Import mydb.sql @ Navicat
3)Open Eclipse and use the right path of MyAuctions @ the directory
4)Run Sign In.java found in the src folder


In order to connect with administrator rights : 
For security reasons, admin accounts have to be created through the database.
Admin account --> username: admin , password: root
